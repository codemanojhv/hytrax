# Changelog

All notable changes to Hytrax are recorded here.

## [0.1.0-alpha.11] - 2026-07-16

- Made Hytrax explicitly user-triggered instead of mandatory during normal coding.
- Added provider-neutral skill installation and stale-skill updates.
- Improved handoff validation, metadata overrides, error messages, and deterministic resume.
- Removed the package self-dependency and upgraded the test toolchain.
- Added cross-agent architecture documentation and regression coverage.

## [0.1.0-alpha.10]

- Added portable handoff and resume commands for cross-agent context transfer.
